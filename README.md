# 🌌 曜羽生態系 (Yaoyu Autonomous Ecosystem)

**曜羽生態系 (Yaoyu Ecosystem)** 是一個完全基於本地運作、具備跨域辯論與自我演化能力的「全自主演化 AI 框架」。它融合了「天地人」三維架構，透過本地大語言模型(如 Ollama) 進行內部靈感議會，並結合實體沙盒 (Windows Sandbox) 驗證生成程式，最後將進化報告投射至多終端代理 (如 AnythingLLM、NotebookLM、ChatGPT、Claude)。

> ⚠️ **注意：本生態系開源版本已嚴格剔除所有與《天堂1》、《天堂2》及任何特定遊戲相關的專案模塊與數據。本套件僅為純粹的 AI 自主推演與協作框架核心。**

## 🚀 系統架構簡介
- **01_核心智能部 (`autonomous_ignition.py`, `inner_council_engine.py`)**：負責驅動生態系的循環與 AI 角色的跨域辯論 (知幾、泠川、艾芯)。
- **02_SOUL_AND_MEMORY**：AI 維度記憶與切片的存放區。
- **03_COMMUNICATIONS (`multi_tab_diplomat.py`)**：自動化抗指紋多終端外交協議，用於同步最新推演結果。
- **05_維度監測部 (`sandbox_injector.py`)**：呼叫無痕 Windows Sandbox 作為代碼試錯的隔離檢疫區。
